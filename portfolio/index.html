About

With 6+ years of professional experience, I've focused my career on building scalable, maintainable, user-centric production systems. I've worked across early-stage startups and established tech firms, blending engineering rigor with creative problem-solving.

I specialize in the TypeScript ecosystem — React and React Native for cross-platform UI, Node.js for backends, and LLM-based agent integrations via Vercel AI SDK and Mastra. I've built component-oriented services in distributed, high-scale systems — asynchronous messaging with RabbitMQ and AWS SQS/SNS, PostgreSQL modeling and query optimization, and observability through structured logging and metrics — shipped with Docker and CI/CD. I believe in SOLID, Clean Code, automated testing, and the idea that great software is as much about the people who use it as the code beneath it.

Based in São Paulo, Brazil. Remote-first, collaborating with teams across LATAM and Europe.
